# three.js-tutorial

Three.js를 단계별로 배우는 튜토리얼 프로젝트입니다.  
웹에서 3D 그래픽을 직접 구현해보며, WebGL과 Three.js의 핵심 개념을 익힙니다.

---

## 목표

- Three.js의 기본구조를 직접 구현하며 이해하기
- 지오메트리, 씬그래프, 재질, 광원, 카메라, 그림자 등의 개념 익히기
